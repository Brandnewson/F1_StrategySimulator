\chapter{Introduction and Background Research}
\label{chapter1}

\section{Introduction}

In multi-agent reinforcement learning (MARL), agents that learn alongside each other do not just solve a task. They adapt to one another. Learning stability and emergent strategy both depend on how incentives are structured between agents, and that dependency is not well characterised. Much of MARL has historically focused on fully cooperative or fully competitive extremes, with many methods tailored to those cases. The partially-aligned or general-sum regime is an emerging research area that remains less mature.

The question this project investigates is whether shared incentive between concurrently learning agents produces genuine cooperative advantage, or merely redistributes competitive outcomes. When two agents race head-to-head, one agent's gain is the other's loss by definition, so no reward formula can produce net joint improvement. That constraint lifts the moment a common adversary is introduced, because both agents can then improve simultaneously by outperforming the third party. Whether shared incentive actually exploits that opening, and how far it can be pushed before it destabilises learning rather than improving, is underexplored in existing literature.

Formula 1 overtaking zone decisions are a well-suited environment to study this. Decisions are discrete and repeated, outcomes are probabilistic given the same zone and gap, and the opponent adapts across laps. The game structure is also adjustable by design. Two agents racing head-to-head form a zero-sum game. Introducing a fixed common adversary breaks that constraint and allows both agents to improve simultaneously. The same simulator, agents, and reward formula can be tested across both structures, isolating game structure as an independent variable. 

To answer this, the project builds a F1-inspired simulator and trains reinforcement learning (RL) agents across zero-sum and non-zero-sum game structures, with the reward-sharing coefficient $\alpha$ varied systematically from pure competition to pure cooperation. These questions draw on three fields: motorsport simulation, RL in competitive environments, and multi-agent incentive design.

\subsection{Motorsport simulation}

The earliest motorsport strategy research treats the simulator as infrastructure for strategy evaluation rather than algorithm comparison. A lap-wise simulation was built modelling tyre degradation, fuel effects, pit stops, and overtaking processes from publicly accessible race data \citep{heilmeier2018racesim}. The goal was to find a good strategy for a given race, which is a different problem from comparing how learning algorithms develop strategies under controlled conditions. This work was extended with neural network surrogates showing that data-driven approximations can accelerate strategy exploration \citep{heilmeier2020virtual}, and online planning for Formula 1 confirmed that planning methods can produce competitive decisions in dynamic conditions \citep{piccinotti2021online}.

The Formula E literature narrows the gap. Multi-car extensions of RL-based race strategy showed that multiple concurrently learning agents can produce competitive results in motorsport \citep{liu2024formulae}, though how incentive structure between agents shapes the strategies that emerge remains open. Learning-based multi-agent race strategies in Formula 1 have also been reported \citep{fieni2026learning}.

Across this body of work, abstracted simulators produce meaningful tactical insight without full physical fidelity, and reinforcement learning is an established framework for racing strategy. No existing motorsport study has systematically varied the incentive structure between concurrent learning agents, or tested how reward alignment between agents changes the strategies that emerge.

\subsection{Reinforcement learning in competitive environments}

Reinforcement learning frames decision-making as an agent that observes a state, selects an action, and receives a reward. Across repeated episodes the agent adjusts its policy, the mapping from states to actions, to maximise long-term reward. Value-based methods estimate a Q-value for each action in each state, and select the action with the highest Q-value at each step \citep{mnih2015human}.

Applied to competitive settings, the framework has produced strong results over the past decade, from board games \citep{silver2018alphazero} to real-time strategy at scale \citep{berner2019dota2, vinyals2019alphastar}. These systems span the three substrate families that dominate modern RL. Value-based methods estimate Q-values and pick the highest-valued action, which suits small discrete action spaces. Policy-based methods learn a policy directly and scale well to continuous control, though they are less sample-efficient. Actor-critic combines both approaches and has become the default for continuous and large-scale domains. Discrete decisions with a modest compute budget suit value-based learning. Within that family, the Deep Q-Network (DQN) was the first to scale value-based learning to high-dimensional state spaces using deep neural networks \citep{mnih2015human}. Section~\ref{sec:dqn-foundations} returns to DQN with the specific requirements of this project in view and selects a variant for the experimental matrix.


The central challenge in competitive RL is that a policy learned against a fixed opponent degrades when that opponent adapts. Self-play addresses this by training an agent against increasingly skilled versions of itself, though it introduces failure modes including cyclic policy dynamics and exploitability by novel opponents. It remains a single-agent method adapted to adversarial settings, leaving open the setting where multiple agents train concurrently.

\subsection{Multi-agent incentive design}

In multi-agent settings, outcomes depend on joint actions rather than individual ones, and agents observe only local state \citep{busoniu2008survey}. Because all agents update at the same time, the environment any one agent experiences keeps shifting as the others' policies change. This effect, called non-stationarity, is the central technical challenge in multi-agent learning \citep{hernandezleal2019survey}.

Mixed-motive settings, where agents' interests are partially aligned and partially conflicting, remain less developed than the pure cooperative and zero-sum cases in both benchmarking and realistic domains \citep{du2023cooperation}. In motorsport, this space is only recently emerging \citep{fieni2026learning}.

Independent learners are the natural first baseline. Each agent stores past experience in a replay buffer and samples from it to update its own policy. When the opponent's policy changes, stored experience from an earlier version of that opponent misdirects updates, and learning can become unstable \citep{foerster2017stabilising, matignon2012independent}. This approach can still outperform centralised alternatives at low agent counts \citep{tan1993multiagent}, but both non-stationarity and the difficulty of assigning credit for shared outcomes worsen as more agents are added \citep{agogino2004unifying, zhao2025maca}.

Credit assignment is the problem of distributing a shared team reward among agents in proportion to their individual contribution, and it remains unsolved in team reward settings \citep{panait2005cooperative, ningxie2024marl}. Two broad families of solutions exist at the level of the reward signal. The first mixes individual and team rewards to balance credit assignment against cooperative alignment \citep{wang2022irat}, which is the mechanism the $\alpha$ coefficient implements. The second gives each agent a reward based on how the team would have performed in its absence. This counterfactual subtraction isolates an agent's individual contribution, and difference rewards are the canonical form of this approach \citep{wolpert2001optimal, devlin2014potential}. Exact computation is impractical at scale, so the team's observed mean is often substituted as a stand-in for the counterfactual \citep{yang2018meanfield}. The two are not the same quantity, and conflating them can lose the alignment property that difference rewards rely on \citep{devlin2014potential}.

Two failure modes arise in this space. Training-order collapse happens when cooperative reward is introduced before individual policies have stabilised, overwriting competent baselines with passive behaviour \citep{matignon2012independent}. Credit-assignment failure happens when overlapping individual contributions prevent team reward from producing useful per-agent gradients \citep{panait2005cooperative, rashid2018qmix}. Telling them apart in practice requires varying incentive alignment across training rather than holding it fixed. Another way of attacking the same credit-assignment problem is at the level of training setup rather than reward signal, through centralised training with decentralised execution \citep{rashid2018qmix}.

The field has mapped each piece in isolation. How they interact under independent concurrent learning, across reward-signal design, agent count, and game structure, remains less well understood.

\section{The niche: concurrent learners under varied incentive alignment}
\label{sec:niche}

Multi-agent learning has been studied most extensively at the two ends of the incentive spectrum. Fully cooperative settings have produced a mature literature on credit assignment and centralised training. Fully competitive settings, especially two-player zero-sum games, have produced strong methods and theory around opponent adaptation, self-play, and equilibrium-oriented learning. The mixed-motive regime between them is the space this project occupies.

Each of the fields surveyed earlier engages with part of this regime. Competitive RL has developed strong methods for training agents that adapt to opponents, though self-play approaches train primarily against past or concurrent versions of the same agent rather than against a second agent with its own learning dynamics. Multi-agent research has characterised non-stationarity, credit assignment, and reward-mixing, typically studying each mechanism in isolation. Motorsport simulation research has shown that abstracted race simulators can produce controlled tactical results without full physical fidelity, which makes an F1-inspired race a practical testbed for studying the three factors together.

\section{The research gap and this project's contribution}

Studies of the partially aligned regime typically isolate one of these factors at a time. What is less well understood is their joint behaviour when all three are varied together on the same learning substrate, with diagnostics that distinguish genuine cooperative advantage from redistribution of competitive outcomes.

This project builds the apparatus that can answer that joint question. The simulator is an F1-inspired race where game structure is switched by the presence or absence of a common adversary, and agent count is configurable from two to five learning agents. A DQN-family substrate is selected to give stable single-agent learning before any multi-agent phase begins, so that algorithm-level differences do not confound the incentive-structure comparisons. The experimental programme sweeps the reward-sharing coefficient $\alpha$ across both game structures and the agent-count range, and evaluates alternative cooperative reward formulations against the $\alpha$-mixed baseline. What the project produces is an empirical characterisation of how reward sharing, game structure, and agent count jointly shape emergent strategy and learning stability.

\section{DQN substrate as the measurement apparatus}
\label{sec:dqn-foundations}

The project's central measurement is whether changes in emergent strategy can be attributed to incentive structure, which requires controlling for algorithm-level differences \citep{henderson2018deep, agarwal2021statistical}. The substrate needs variants that are small and well-characterised, a decision space that matches discrete action selection at a small number of zones, and off-policy learning so rare overtaking events can be revisited during training \citep{mnih2015human, schaul2016prioritized}.

The DQN family has this structure by design. Each variant in the lineage was introduced to correct a named failure mode of the previous one. Double DQN targets Q-value overestimation under noisy rewards \citep{vanhasselt2016double}. Dueling DQN separates state value from action advantage to stabilise learning where the choice of action has little effect on the outcome \citep{wang2016dueling}. Prioritised experience replay addresses uniform sampling when informative transitions are rare \citep{schaul2016prioritized}. Rainbow combines these into a single algorithm \citep{hessel2018rainbow}. Because each mechanism has a clear target, differences between variants map onto specific known biases rather than unrelated implementation choices. Independent Q-learning, the standard value-based MARL baseline \citep{rashid2018qmix}, places the family within the multi-agent literature this project draws on.

The comparison set is therefore vanilla DQN, Double DQN, Dueling DQN, and a Rainbow-style variant combining the three with prioritised replay. Chapter~\ref{chapter2} sets out the experimental methodology for these comparisons, and Chapter~\ref{chapter3} covers their implementation.


\section{Aim, objectives, and research questions}

\subsection{Aim}

Multi-agent learning has been studied extensively in fully cooperative and fully competitive settings. The space between, where self-interested agents are only partially aligned, is recognised in the literature but remains less mature in its benchmarks, evaluation metrics, and realistic testbeds. The aim of this project is to characterise how cooperation between self-interested learning agents changes as incentive alignment and agent count are varied, and to identify the mechanisms by which cooperation succeeds, fails, or breaks down.

\newpage
\subsection{Research questions and hypotheses}
\label{sec:research-questions}
\begin{description}
    \item[RQ1.] What strategic behaviour emerges when independent reinforcement learning agents train concurrently under pure competition?
    \item[H1.] Concurrent training produces differentiated zone coverage and opposing risk profiles through iterated best-response dynamics. No explicit coordination mechanism is required.

    \item[RQ2.] How does varying the reward-sharing coefficient $\alpha$ alter emergent strategy and learning stability in two-agent zero-sum and non-zero-sum settings, and can curriculum scheduling recover cooperative performance when fixed $\alpha$ fails?
    \item[H2.] Above a threshold, sharing too much reward turns successful overtakes into a net penalty and both agents drift into passive play. In non-zero-sum races with a common rival, partial sharing helps up to that same threshold and then starts to hurt. Curriculum scheduling recovers some of the lost performance.

    \item[RQ3.] How many agents can a non-zero-sum race hold before cooperation stops working, and does gradual scheduling change the answer?
    \item[H3.] Beyond a certain agent count, no value of $\alpha$ and no training schedule brings cooperation back, because the problem is credit assignment rather than training order.

    \item[RQ4.] Do alternative cooperative reward formulations preserve the advantage that $\alpha$-based mixing produces?
    \item[H4.] The mean-field substitute for the true counterfactual inverts the cooperative signal, so the advantage observed under $\alpha$-based mixing will not reproduce under this approximation.
\end{description}

\subsection{Objectives}
\begin{itemize}
    \item Build a racing simulator with configurable game structure, agent count, and diagnostic output that distinguishes cooperative advantage from redistributed competitive outcomes.
    \item Pick a DQN variant that trains stably in the single-agent setting to act as a controlled substrate for the comparisons that follow.
    \item Measure how the reward-sharing coefficient $\alpha$ affects emergent strategy and learning stability across zero-sum and non-zero-sum game structures.
    \item Test whether cooperative performance breaks down as agent count grows, and whether curriculum scheduling can recover it.
    \item Evaluate alternative cooperative reward formulations, and check whether a zero-shot language model with domain reasoning can match trained agents.
\end{itemize}
