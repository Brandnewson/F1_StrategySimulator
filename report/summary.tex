This dissertation investigates reinforcement learning for competitive race strategy in an F1 inspired simulator. The focus is reproducible comparison rather than a single best score.

Low complexity experiments are run first. In this mode one DQN driver races one base driver.

The comparison includes vanilla DQN and double DQN and dueling DQN and rainbow lite DQN under fixed budgets. Matched seeds keep each comparison fair.

Confidence intervals and behaviour diagnostics are reported for overtaking decisions. These outputs explain performance differences with clear evidence.

Medium complexity and high complexity are kept for future stages. This choice supports incremental development with reliable analysis.
