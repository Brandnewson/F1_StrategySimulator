This project investigates how incentive structure shapes emergent strategy when multiple reinforcement learning agents train together in a stochastic environment. The experimental domain is Formula 1 race strategy. The central variable is a reward-sharing coefficient that controls how much each agent values a teammate's performance alongside its own, ranging from purely self-interested to fully cooperative. By holding everything else constant and varying this coefficient, we isolate the effect of incentive design on the strategies agents develop.

Four variants of the Deep Q-Network family are compared as the base algorithm. In competitive settings, algorithm choice determines whether stable strategic behaviour emerges at all. Weaker variants collapse into passivity in close to half of all trials. The strongest variant, combining prioritised experience replay with double-target correction, avoids collapse entirely and becomes the stable foundation for all subsequent experiments.

Introducing cooperative reward into a purely competitive game produces a sharp breakdown rather than a gradual one. Below a certain threshold, partial alignment does not destabilise learning. Above it, agents converge to passivity in the majority of trials. Changing the game structure so that both agents can improve simultaneously resolves this. With a shared adversary to outperform together, cooperative incentive produces a large improvement in joint performance. An ablation study confirms the gain comes from the reward signal itself rather than from the change in game structure.

Scaling to more agents reverses the result. The cooperative advantage that appears in a three-agent setting disappears as team size grows, with the transition occurring abruptly at four agents. Curriculum training, which establishes individual policies before introducing cooperative reward, recovers much of the lost performance. This reveals two distinct failure modes. One arises from training order and can be resolved by scheduling. The other is a deeper credit assignment problem that requires architectural change beyond the independent learning framework.

A large language model given the race state as a natural language description with no prior training matches the best-trained agent across all tested conditions. The strategies are qualitatively different but arrive at similar outcomes, which suggests the environment is constrained enough for domain reasoning to approximate what the agent learned. A theoretically motivated reward formulation designed to credit each agent only for its own contribution fails both analytically and empirically. The approximation used inverts the cooperative gradient, penalising agents for their teammates' successes rather than rewarding their own actions.

The primary finding is that incentive design and agent count jointly determine whether cooperation produces advantage or harm, independently of algorithm choice. In small teams, careful alignment of incentives produces real cooperative gains. In larger teams, reward formula design alone is not sufficient and architectural changes become necessary.
